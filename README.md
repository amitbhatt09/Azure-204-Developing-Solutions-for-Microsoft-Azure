# AZ-204: Developing Solutions for Microsoft Azure – Revision Guide

## Overview

This repository contains my **concise revision notes, preparation strategy, and important resources** for the **AZ-204: Developing Solutions for Microsoft Azure certification**.

The purpose of this repository is to:

- Provide **quick revision notes for Azure developers**
- Cover **all major AZ-204 exam domains**
- Highlight **frequently asked exam topics**
- Share **learning resources and preparation strategy**

This repository can also help developers preparing for **Azure Developer interviews**.

---

## About the AZ-204 Certification

The **AZ-204 certification** validates the ability to design, build, test, and maintain cloud applications using Microsoft Azure.

### Skills measured in the exam

- Develop **Azure compute solutions**
- Develop for **Azure storage**
- Implement **Azure security**
- Monitor and troubleshoot solutions
- Connect to **Azure and third-party services**

Official certification page:

https://learn.microsoft.com/en-us/certifications/azure-developer/

---

## My Preparation Strategy

My preparation approach was divided into **three phases**.

### Phase 1 – Understanding Azure Services

First, I focused on understanding how different Azure services work and interact with each other.

Key services studied:

- Azure App Service
- Azure Functions
- Azure Storage
- Azure Service Bus
- Azure Event Grid
- Azure Key Vault

---

### Phase 2 – Hands-on Practice

Hands-on practice is **very important for AZ-204**.

Activities performed:

- Deploying applications using **Azure App Service**
- Creating **Azure Functions**
- Working with **Blob Storage and Queue Storage**
- Implementing authentication using **Azure AD**
- Monitoring applications using **Application Insights**

---

### Phase 3 – Revision and Practice Tests

The final phase focused on:

- Revising important concepts
- Practicing scenario-based questions
- Reviewing Microsoft Learn modules
